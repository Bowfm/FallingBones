# FallingBones
 Major assignment for Advanced Programming

# **GAME: FALLING BONES**

--- INSTRUCTIONS ---

Use **LEFTARROW (<-)** and **RIGHTARROW (->)** to move the basket around to catch bones falling from the sky

**CLICK PAUSE BUTTON** or **PRESS (P)** to pause/resume game
